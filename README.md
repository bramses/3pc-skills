# 3pc-skills

Open-source skills directory for [3PC Labs](https://github.com/bramses/3pc-skills). Each skill lives in its own directory under [`skills/`](skills/).

## Skills

| Skill | What it does |
| --- | --- |
| [3pc-labs](https://github.com/bramses/3pc-skills/tree/main/skills/3pc-labs) | Reference knowledge base about 3PC Labs, an AI consulting organization that runs workshops and builds custom AI tools. |
| [update-readme](https://github.com/bramses/3pc-skills/tree/main/skills/update-readme) | Update and improve a project's README.md by reading the actual codebase, comparing it against the Awesome README list (https://github.com/matiassingers/awesome-readme), and iterating conversationally with the user before touching any files. |
